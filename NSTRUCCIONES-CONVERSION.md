# 📊 Instrucciones para Convertir a PowerPoint

## 🎯 Objetivo
Convertir los archivos Markdown (.md) a presentaciones PowerPoint (.pptx) profesionales

---

## 🔧 Método 1: Usar Marp (RECOMENDADO - Más Fácil)

### **¿Qué es Marp?**
Marp convierte Markdown directamente a PowerPoint con diseño profesional.

### **Paso 1: Instalar Marp CLI**

#### **En Windows:**
```bash
# Abre Terminal y ejecuta:
brew install marp-cli
# O si usas npm:
npm install -g @marp-team/marp-cli
