## 🎯 Objetivos de Aprendizaje

Al finalizar esta formación, los operarios serán capaces de:

✅ Identificar peligros en el área de trabajo
✅ Utilizar correctamente los Equipos de Protección Personal (EPP)
✅ Operar máquinas de mecanizado por viruta y abrasión de forma segura
✅ Realizar labores de reparación industrial con protocolos de seguridad
✅ Responder ante emergencias siguiendo procedimientos establecidos
✅ Cumplir con legislación y normativas vigentes
